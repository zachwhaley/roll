# Roll

Dice rolling programs written in various languages

## C

```
$ gcc -std=c11 roll.c -o roll
$ ./roll d6
.---.
| 2 |
'---'

2
```

## Go

```
$ go build roll.go
$ ./roll 2d4
  .
 / \
/ 2 \
`---´

  .
 / \
/ 1 \
`---´

3
```
